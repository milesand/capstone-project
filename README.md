# Capstone-project

<body>
<h4>Capstone 폴더에서 python manage.py runserver를 실행한 뒤, 주소창에서 localhost:8000으로 진입하면 됩니다.</h4>

- /register - 회원가입
- /login - 로그인
- /users - 가입한 사용자들의 프로파일 출력

회원가입을 하면 등록한 이메일로 인증 URL이 보내지며, 이 URL을 클릭해서 인증을 수행할 수 있습니다. 인증을 완료하면 로그인이 가능합니다.

필요한 모듈
- django
- pythonrestframework
- python-rest-knox
</body>
